COMMEMI 3D-2A synthesize time series data.

File:

*.TBL *.TS3 *.TS4 *.TS5
	Data in Phoenix MTU-5A format.
4884.CLB MT8C9003.CLC
	Calibration files.
	
*.STS1 *.STS2 *.STS3(Compressed file in Zip format)
	The same time series data in text format with absolute field values.
	Each column is: UTC time, Ex, Ey, Hx, Hy, Hz.
	Units:mV/km,nT.
	
Samplerate:
	*.TS3 *.STS1 2400Hz
	*.TS4 *.STS2 150Hz
	*.TS5 *.STS3 15Hz
