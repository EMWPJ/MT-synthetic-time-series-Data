One-dimensional models synthesize time series data.
Model parameters: three layers. The resistivity is 10 ohms, 100 ohms, 0.1 ohms. The layer thickness is 10km and 20km.

File:

MT1D.TBL MT1D.TS3 MT1D.TS4 MT1D.TS5
	Data in Phoenix MTU-5A format.
4884.CLB MT8C9003.CLC
	Calibration files.
	
MT1D.STS1 MT1D.STS2 MT1D.STS3(Compressed file in Zip format)
	The same time series data in text format with absolute field values.
	Each column is: UTC time, Ex, Ey, Hx, Hy, Hz.
	Units:mV/km,nT.
	
Samplerate:
	MT1D.TS3 MT1D.STS1 2400Hz
	MT1D.TS4 MT1D.STS2 150Hz
	MT1D.TS5 MT1D.STS3 15Hz
